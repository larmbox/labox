Labox

Labox is a Vue 3 component library.

Get Started →

Keeping it simple

Labox provides commonly used components along with an easy-to-understand API.

Dynamic themes

Labox does not push any specific design and only provides the base components. You are in charge of the look and feel.

Ready for SSR

Labox works out of the box with server-side rendering. Nuxt module coming soon!
